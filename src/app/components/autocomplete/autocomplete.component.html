Autocomplete

O Autocomplete permite que o usuário observe opções sobre o conteúdo relacionado ao campo.

Com dois atributos

Com loadmore

Atributo Função Atributo modelValue Guarda o valor do item selecionado Atributo list Array ou array de objetos que contem as opções do select loading Utilizado para controlar a barra de carregamento do select Atributo select Dispara um evento após a seleção de um item. Atributo loadMore Evento disparado quando o scroll do autocomplete chega perto do final da barra de rolagem Atributo theme-class Utilizado para definir o tema do autocomplete Atributo primary* Utilizado para definir o item principal do autocomplete secondary* Utilizado para definir o item secundario do autocomplete * O atributo list deve ser um array de objetos
